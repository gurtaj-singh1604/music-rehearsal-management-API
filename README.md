# Music Rehearsal Management API

This is my Capstone Project for Milestone 1.

The project is a back-end RESTful API built with Node.js, TypeScript, Express, and Firebase Firestore. It is designed to help manage songs, setlists, and rehearsals for a music rehearsal workflow.

## Features

- Songs CRUD
- Setlists CRUD
- Rehearsals CRUD
- Firebase Firestore integration
- Swagger / OpenAPI documentation
- Jest and Supertest testing
- Input validation with Joi
- Layered architecture using routes, controllers, services, and repositories

## Tech Stack

- Node.js
- TypeScript
- Express
- Firebase Firestore
- Joi
- Swagger UI Express
- Swagger JSDoc
- Jest
- Supertest
